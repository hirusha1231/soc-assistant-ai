# 🔐 SOC Analyst Assistant - AI-Powered ChatBot

An intelligent AI assistant for Security Operations Center (SOC) analysts that answers security alerts and provides incident response guidance using trained playbooks.

## ✨ Features

- **Alert Translation**: Converts raw security alerts into plain English explanations
- **Playbook Response**: Provides step-by-step incident response based on security playbooks
- **MITRE ATT&CK Mapping**: Maps alerts to MITRE ATT&CK techniques
- **RAG Implementation**: Uses Retrieval-Augmented Generation for accurate responses
- **Web Interface**: Clean Streamlit UI for easy interaction
- **Quick Questions**: Pre-built questions for common scenarios
